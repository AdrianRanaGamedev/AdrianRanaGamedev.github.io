SHIELD

OPERATION: NEW BLOOD

As part of a Shield-themed escape room, players will be recruits of the agency and will have to face a series of tests to prove their worth as valid agents. One of the tests is a virtual shooting range that they will have to pass to get a code necessary to escape.

  • Multiplayer game mechanics
  • Shooting mechanics and scoring
  • Placement of decals and particle systems for feedback
  • Programming and implementation of diegetic interface on screens within the game
  • Management of animations using Unity's Mecanim animation system

MAP GAME

The players are tasked with finding villains of the Marvel universe on a global map using clues found within the escape room. The game features a detailed map of the world, with locations of villains marked as locations to be searched. Players must use their problem-solving skills and teamwork to find the villains before time runs out. As they progress through the game, players will uncover hints and clues that will help them narrow down their search and ultimately find the villains.

UNITY C# UI PC

BOMB DEFUSAL

This application is located in a special briefcase prop in an escape room and is connected to a keyboard via an Arduino. The application displays a countdown timer for a bomb that the players must defuse in order to progress in the escape room. The screen also shows the code that the players input. The players must input the correct code to defuse the bomb. Half of the code is obtained in a VR game and the other half in a map game. The players have to combine the codes to defuse the bomb and continue in the escape room.
