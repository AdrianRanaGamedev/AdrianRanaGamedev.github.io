CSI:NY

For a CSI: New York-themed escape room, we programmed a police application for players to investigate a crime. The computer application is the center of the escape room. Players have to find clues, scan them on special machines that communicate with the police application, unlocking clues that advance the investigation and unlocking new information.


My responsibilities included:

  • Programming puzzle games
  • Programming connection via MQTT to custom machines in the escape room
  • Improve the game feel
