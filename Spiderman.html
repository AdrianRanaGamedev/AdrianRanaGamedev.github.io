SPIDER-MAN

REPTILIA

VR LAB EXPERIMENT

The final challenge of this Spiderman-themed escape room is a seemingly empty room with virtual reality glasses on a table. Once the players put them on, the room transforms into a laboratory with a comic book aesthetic. In this multiplayer game, players have to collaborate to solve a series of puzzles. Some of the mechanics implemented in this game include:

  • Picking up, throwing, and passing objects between players
  • Pressing buttons in a specific sequence
  • Conducting a laboratory experiment
  • Dynamic feedback using particle systems
  • Interactions with the enviroment
  • Spiderman animations

AR TABLET APPLICATION

We developed a tablet application using Unity and Vuforia. Players have to use the application on the tablet to communicate with Peter Parker through instant messaging, receive hints, and use the camera to search for clues using augmented reality.
